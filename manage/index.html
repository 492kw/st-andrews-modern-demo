How this would work for Damon

An office editor changes a sermon, event or page text here, checks the preview, then chooses Publish. The public website stays static and available while that happens. If publishing fails, the previous approved content stays live and the editor can try again.

Replacement point: assets/storage-adapter.js is the single place to swap demo browser storage for a Git-backed CMS or a Cloudflare content adapter. The page design does not need to change.
